Tilburg Science Hub is a platform helping individual researchers, data scientists, and their teams to efficiently work on empirical research projects.

We provide information about workflow and data management and tutorials that teach researchers how to organize and document their data and code so the research becomes sustainable and reproducible. This, in turn, leads to time savings and transparency in the process.

The platform is an open source initiative which means that anybody can contribute or suggest improvements. This can be done in two ways, either via a GitHub Issue or via a GitHub Pull Request.

Github Issue
If you want to suggest a change for the website you can do this by creating a GitHub Issue. You can, for example, bring a bug to our attention or point out a typo. You can also create an issue to suggest a new feature or to ask for new content to be added. For more information visit the contribute page
